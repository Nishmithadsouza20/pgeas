## How to take screenshots on Windows

- **Snipping Tool** — press `Win + Shift + S`, drag to select, paste into Paint and save as PNG
- **Full page** — press `PrtScn` or use browser DevTools → Ctrl+Shift+P → "Capture screenshot"
- **Recommended size** — 1280×800 browser window for consistent screenshots
